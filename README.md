# Vulnerability Assessment & Management System

A Java-based cybersecurity management application developed using Object-Oriented Programming concepts. The project simulates the workflow of a vulnerability assessment platform by managing client information, target systems, vulnerability records, CVE identifiers, severity ratings, CVSS scores, and patch statuses.

## Features

* Client information management
* Target system tracking
* Vulnerability record management
* CVE and CVSS score handling
* Patch status monitoring
* Console-based reporting system

## Technologies Used

* Java
* Object-Oriented Programming (OOP)

## Learning Objectives

This project was developed to strengthen understanding of Java classes, objects, encapsulation, methods, and real-world cybersecurity data modeling.
